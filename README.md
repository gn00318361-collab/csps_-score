# 🏆 青溪國小 114 學年度親職教育日 — 園遊會攤位評分系統

供評分老師於園遊會當天，透過手機即時為各班攤位打分數，管理者可即時查看統計結果。

🔗 **線上預覽 (Live Demo):** [https://gn00318361-collab.github.io/csps_-score/](https://gn00318361-collab.github.io/csps_-score/)

---

## ✨ 特色功能 (Features)

- **📱 行動裝置優先**：專為手機操作設計，評分老師現場即可使用。
- **👩‍🏫 評分老師模式**：選擇年級與姓名後，逐班進行評分並即時送出。
- **📊 管理者模式**：輸入密碼後可查看各班得分、統計排名結果。
- **🔒 身份驗證**：評分老師依年級分配，管理者需密碼登入。
- **⚡ 純前端架構**：無需後端伺服器，GitHub Pages 即可部署。

## 🛠️ 技術 (Tech Stack)

- **JavaScript (Vanilla)**：處理評分邏輯、身份選擇與分數統計。
